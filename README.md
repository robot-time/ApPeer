# ApPeer

An iOS social networking app built with SwiftUI and Firebase.

## Features
- Peer connections and social feeds
- Firebase Authentication and Firestore backend
- Real-time updates and community building
- Clean, native iOS UI built in SwiftUI

## Tech Stack
- SwiftUI
- Firebase (Auth, Firestore)
- CocoaPods

## Getting Started
1. Clone the repo
2. Run `pod install` in the project directory
3. Open `APeer.xcworkspace` in Xcode
4. Build and run on a simulator or device

## Status
Early prototype — initial commit with core architecture in place.
